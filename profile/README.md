## 🧭 About Xeze

Xeze is an open-source organization focused on building developer infrastructure, AI-powered tooling, and privacy-centric applications. We ship production-ready tools in **Go**, **Python**, and **TypeScript** — designed to be self-hosted, secure, and easy to extend.

## 🤝 Contributing

We welcome contributions of all kinds — bug fixes, new features, documentation, and ideas.

1. Fork the repo you want to contribute to
2. Create a feature branch: `git checkout -b feat/your-feature`
3. Commit your changes and open a pull request
4. Describe what you changed and why

For major changes, open an issue first to discuss the approach.
